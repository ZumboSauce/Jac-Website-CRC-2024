Tutorial

Nous avons réalisé un tutoriel sur comment faire une mural pour la catégorie "kiosque" pour mettre en valeur l'ingéniosité de cette catégorie. Il faut beaucoup d'énergie et utililié plusieurs techniques astucieuses pour créer murales qui incarnent l'ambiance du Kiosque. Ce tutoriel présente les étapes de base pour créer une murale.
